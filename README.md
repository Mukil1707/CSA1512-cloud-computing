# SkillHire Cloud Resume-to-Job Matcher Using NLP Skill Gap Intelligence

## Project Overview

SkillHire is a cloud-based Resume-to-Job Matching system that uses Natural Language Processing (NLP) and Skill Gap Intelligence to connect job seekers with suitable job opportunities. The system analyzes resumes, extracts technical and soft skills, compares them with job requirements, identifies missing skills, and recommends the most suitable jobs.

This project demonstrates how cloud computing concepts are integrated into an AI-powered recruitment platform.

---

## Objective

- Match resumes with relevant job openings.
- Extract skills using NLP.
- Identify skill gaps between resumes and job descriptions.
- Recommend jobs based on matching scores.
- Generate analytics for recruiters and administrators.

## System Workflow

1. User uploads a resume.
2. Resume is sent to the cloud server.
3. NLP engine parses the resume.
4. Skills are extracted from the resume.
5. Skills are compared with job requirements.
6. Skill gaps are identified.
7. Matching score is calculated.
8. Recommended jobs are displayed.
9. Results are stored in the cloud database.
10. Analytics dashboard is updated.
